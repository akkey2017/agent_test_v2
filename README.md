# 🐍 スネークゲーム (Snake Game)

シンプルで楽しいスネークゲームです！

## 🎮 遊び方

1. `index.html` をブラウザで開く
2. 「ゲームスタート」ボタンをクリック
3. スネークを操作してりんごを食べよう！

### 操作方法

- **キーボード**: 矢印キー (↑↓←→) または WASD
- **スマホ/タブレット**: 画面下のボタンをタップ

## ✨ 機能

- 🍎 りんごを食べるとスコアアップ！
- ⚡ スコアが上がるとスピードアップ！
- 🏆 ハイスコアはブラウザに保存
- 📱 スマホ対応

## 🚀 プレイ

[GitHub Pages でプレイ](https://akkey2017.github.io/agent_test_v2/)

または、ローカルで `index.html` を開いてプレイできます。

## 🎯 ルール

- 壁にぶつかるとゲームオーバー
- 自分の体にぶつかるとゲームオーバー
- できるだけ長く生き残ってハイスコアを目指そう！
